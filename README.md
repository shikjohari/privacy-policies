# OnlineCalcTools Privacy Policies

This repository is designed to host privacy policies for mobile applications published under the **OnlineCalcTools** brand on the Google Play Store.

## Hosting Instructions (GitHub Pages)

1. **Create a New Repository**: Create a repository named `privacy-policies` on GitHub.
2. **Push Files**: Upload the contents of this directory to that repository.
3. **Enable Pages**:
   - Go to **Settings > Pages**.
   - Set Source to **Deploy from a branch** (main).
   - Click **Save**.

## Live Links for Play Console

Once hosted, use these direct URLs in your Google Play Console:

- **EMI Calculator**: `https://<username>.github.io/privacy-policies/emi-calculator.html`
- **Loan Affordability**: `https://<username>.github.io/privacy-policies/loan-affordability-calculator.html`
- **Landing Page**: `https://<username>.github.io/privacy-policies/`

## Content Structure

- `index.html`: Professional landing page linking to all policies.
- `emi-calculator.html`: Specific policy for EMI Calculator.
- `loan-affordability-calculator.html`: Specific policy for Loan Affordability.
